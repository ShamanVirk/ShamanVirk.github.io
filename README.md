# Ahmad Shaman Virk — Portfolio

Interview-facing portfolio site, deployed on GitHub Pages.

**Live:** https://ShamanVirk.github.io

## Structure

```
index.html          # UI renderer — no hardcoded content
data.js             # All CV content — edit this to update the site
assets/
  Ahmad_Shaman_Virk_CV.pdf   # Downloadable CV
```

## Updating content

Edit `data.js` only. The site renders dynamically from it — no changes to `index.html` needed.

## Updating the CV PDF

Replace `assets/Ahmad_Shaman_Virk_CV.pdf` with the new file (keep the same filename).

## Navigation

- Arrow keys `←` `→`
- On-screen Prev / Next buttons
- Click any dot at the bottom to jump to a slide
